## 1. 개발 범위 및 전략

- **현재 범위:** 1~2학년 수와 연산 (덧셈, 뺄셈)
- **추후 로드맵:** 3~4학년 곱셈/나눗셈 완료 후 → 측정, 데이터, 기하 등 추가
- **설계 원칙:** 나선형 교육과정(Spiral Curriculum) 기반. 확장 가능한 게임은 학년/난이도별 반복 배치, 특정 개념 특화 게임은 1회 배치.

## 8. 문제 생성 규칙 정의

### 20이내 덧셈 (Addition within 20)
- **정의:** 두 수의 합(sum)이 20 이하인 덧셈
- **규칙:** A + B = C 에서 C ≤ 20
- **범위:** A: 1~19, B: 1~19, C(정답): 2~20
- **예시:** 8+6=14 ✅ / 12+9=21 ❌

### 20이내 뺄셈 (Subtraction within 20)
- **정의:** 피감수(minuend)가 20 이하인 뺄셈
- **규칙:** A - B = C 에서 A ≤ 20, A > B
- **범위:** A: 2~20, B: 1~19, C(정답): 0~19
- **예시:** 14-5=9 ✅ / 25-8=17 ❌

### 100이내 덧셈/뺄셈 (동일 패턴 확장)
- **덧셈:** A + B = C 에서 C ≤ 100
- **뺄셈:** A - B = C 에서 A ≤ 100, A > B

## 9. 신규 게임 아이디어

### 1학년용

**Block Builder** (Ones and tens)
상단에 숫자가 제시되면 10묶음 블록과 1개짜리 블록을 드래그해서 자릿값을 표현. 나선형 확장 가능(1학년 ~20, 2학년 ~100).

**Seesaw Smash** (수 비교)
시소 양쪽에 숫자가 올라와 있고, 더 큰 쪽을 터치하면 시소가 기울며 작은 쪽 캐릭터가 날아감. >, <, = 선택 모드 가능. 나선형 확장 가능.

**Flip Card** (덧뺄셈 관계)
카드 앞면에 덧셈식(5+3=8), 뒤집으면 빈칸 뺄셈식(8-3=?). 역연산 관계를 직관적으로 체험. 개념 특화 1회 배치.

### 2학년용

**Rocket Launch** (수 비교)
두 로켓에 숫자가 적혀있고 더 큰 쪽을 터치하면 발사. 연속 정답 시 더 높이 올라가는 보상.

**Train Builder** (100이내 합 만들기)
기차 앞칸에 목표 숫자, 플랫폼의 화물칸 2개를 골라 합을 맞추면 기차 출발. Number Balance의 상위 버전이지만 다른 비주얼.

**Bubble Pop Chain** (여러 수 덧셈)
숫자 비눗방울이 떠다니고, 목표 합에 맞게 3~4개를 연속 터치. 누적 합 실시간 표시.

**Treasure Dig** (100이내 뺄셈)
뺄셈식이 제시되고, 정답 깊이의 삽을 선택해 모래를 파면 보물 발견. Deep Sea Dive와 테마/조작 차별화.
